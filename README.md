###### hikoServer ##########
NodeJS server for handling an android app (Hiko)

how to use in local (you must have mongoDB installed on your computer):
 - git clone https://github.com/theozzz/hikoServer.git
 - npm install
 - launch the mongodb service (mongod)
 - node server.js
